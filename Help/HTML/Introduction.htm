Overview

What is Open Twebst?
Open Twebst is an open source web automation framework for Internet Explorer that can be used within any environment that supports COM, from scripting languages JScript, VBScript to higher level programming languages C#, VB.Net, VBA/Excel, C++.

The framework includes two components: Open Twebst Web Recorder that generates most of the automation code and Open Twebst Library which is a collection of programmable objects that creates a web automation API.

What Open Twebst can do?
  • automate regression tests of web applications
  • automate web actions and data-entry
  • automatically log in to different web sites
  • fill out web-forms automatically
  • extract data from web pages

Open Twebst features
  • Start new browsers and navigate to a specified URL; connect to already existing IE web pages.
  • Automate embedded IE browser control in any application.
  • Simulates user behaviour generating hardware or browser events.
  • Get full programmatic control over HTML elements and frames.
  • Advanced search of browsers and HTML elements using wild-cards.
  • Perform actions on all HTML controls (button, combo-box, list-box, edit-box, upload control).
  • Get access to native IE objects so you don't need to learn new API if you already know IE web programming.

Open Twebst Web Recorder features
  • Generate web automation code using the language of your choice: JScript, VBScript, VB.Net.
  • Record web actions on all HTML web controls (anchor, button, combo-box, list-box, edit-box).
  • Easily create web macros through an intuitive graphical interface.

System requirements

Windows XP, Windows Vista, Windows 8
Internet Explorer 6 to 11 - protected mode ready
